int  body,

)
private

Should a contact be dispatched to an installed listener for some body?

We have installed listeners in mDispatch for bodies. This tests to see if either of the two bodies involved in the collision are in the mDispatch map. If so, the listener reference value is set and a true value is returned.

Parameters
contactContact to test
bodyWhich body for the contact (1 or 2?)
listenerWhere to put the listener to call the function on
